# NS-EasyMetric

EasyMetric은 통계 분석을 쉽고 직관적으로 수행할 수 있는 웹 기반 도구입니다. 다양한 통계 분석 기능을 제공하여 데이터 기반 의사결정을 지원합니다.

## 주요 기능

- 기술통계 분석 (평균, 분산, 표준편차 등)
- 가설검정 및 회귀분석(t-test, ANOVA 등)
- Gage R&R 분석, 공정능력분석 등
- 그래프 시각화(산점도, 교호작용도 등)
